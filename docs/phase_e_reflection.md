# Phase E振り返り: 「隣の人指数」仮説の検証と学び

**作成日**: 2025年12月1日
**作成者**: 関根 禎浩

---

## 1. Phase Eの目的と仮説

### 1.1 背景

Phase Dで「恐怖指数」を用いた分析により、Phase C（COVID-19死亡数）の2.4倍の説明力（R²=5.7%）を達成しました。しかし、依然として説明力は限定的でした。

Phase D補足資料（[phase_d_supplementary.md](phase_d_supplementary.md)）のセクション7で、以下の仮説を立てました：

> 「日本における感染対策の決定要因は、『隣にいる人間がどう行動しているか』である」

### 1.2 Phase Eの核心的仮説

**日本文化特有の社会的同調圧力**に着目：

```
個人の恐怖心 < 周囲の人々の行動
    ↓
マスクをしていない人が増える
    ↓
「自分もしなくていいか」という心理
    ↓
感染対策の緩和
    ↓
インフルエンザの回復
```

**予測**:
- 「隣の人指数」（マスク関連検索）は恐怖指数より強い相関を示す
- 期待されるR²: 0.3～0.5（30～50%の説明力）

### 1.3 測定指標の設計

**Phase D（恐怖指数）** vs **Phase E（隣の人指数）**:

| | Phase D | Phase E |
|---|---|---|
| **測定対象** | 内面的な恐怖心 | 外面的な行動 |
| **キーワード** | 「コロナ 死亡」「医療崩壊」 | 「マスク」「手洗い」 |
| **焦点** | 脅威認知 | 具体的対策行動 |
| **仮説** | 恐怖が行動を促す | 周囲の行動が自分の行動を促す |

**隣の人指数のキーワード選定**:
- マスク（重み 3.0）
- マスク 着用（重み 2.5）
- 手洗い（重み 1.5）
- 消毒（重み 1.5）
- 換気（重み 1.0）

## 2. Phase Eの結果

### 2.1 数値的な結果

| 指標 | 相関係数 | R² | 傾き | ランキング |
|------|---------|-----|------|----------|
| **Phase D（恐怖指数）** | **-0.239** | **0.0571** | **-0.195** | **🥇 1位** |
| Phase C（COVID-19死亡数） | -0.154 | 0.0238 | -0.046 | 🥈 2位 |
| **Phase E（隣の人指数）** | **-0.111** | **0.0122** | **-0.095** | **🥉 3位** |

**予測**: R²=0.3～0.5（30～50%）

→ **予測の24～40倍も低い説明力**

### 2.3 Phase Dとの比較

- 恐怖指数の説明力はPhase Eの**4.7倍**
- 相関係数も恐怖指数の方が**2.2倍強い**

## 3. なぜ仮説は失敗したのか？

### 3.1 仮説1: 測定誤差 - 検索行動≠実際の行動

**問題点**:
- Google検索「マスク」は関心の表れであり、実際のマスク着用率ではない
- 関心が高い時期（2020年初期）に検索が集中
- 習慣化後は検索しなくなる（行動は継続しているが検索しない）

**データパターン**:
```
2020年初期: 検索↑、着用率↑（関心が高い）
2021-2022年: 検索↓、着用率↑（習慣化、検索不要）
2023年: 検索↓、着用率↓（規範の緩和）
```

→ 検索トレンドはマスク着用の「導入期」を捉えるが、「定着期」は捉えられない

### 3.2 仮説2: 「慣れ」による行動の質的変化

**形式的対策 vs 実質的対策**:

| 時期 | マスク着用 | 密の回避 | 手洗い徹底 | 外出自粛 | 総合効果 |
|------|----------|---------|-----------|---------|---------|
| 2020年 | ✅✅✅ | ✅✅✅ | ✅✅✅ | ✅✅✅ | 高 |
| 2021-2022年 | ✅✅ | ✅✅ | ✅ | ✅ | 中 |
| 2023年 | ✅ | ❌ | ❌ | ❌ | 低 |

**観察**:
- マスク着用は長期間継続（検索トレンドは減少）
- しかし、密の回避や徹底的な手洗いは緩和
- 形式は維持、実質は低下

→ **隣の人指数（マスク検索）は「形式」を測定し、「実質」を捉えられない**

### 3.3 仮説3: マスク着用の「意味」の変化

**2020年（恐怖心主導）**:
- 「自分が感染したくない」という恐怖
- 「周りの人を感染させたくない」という責任感
- マスク着用は**内発的動機**

**2021-2022年（社会規範）**:
- 「みんながしているから」という同調
- 「しないと白い目で見られる」という圧力
- マスク着用は**外発的動機**

**2023年（慣習）**:
- 「なんとなく続けている」という惰性
- 「感染対策」としての意識は薄い
- マスク着用は**無意識的習慣**

→ **同じ「マスク着用」でも、時期により意味が変化**

### 3.4 仮説4: 因果の逆転 - 恐怖が先、行動が後

**当初の仮説（Phase E）**:
```
周囲の行動 → 自分の行動 → インフルエンザ抑制
```

**Phase Eの結果が示唆する真の因果**:
```
【Phase D】恐怖指数の上昇（最も重要）
    ↓
包括的な対策行動（マスク、距離、手洗い、外出自粛）
    ↓
インフルエンザ抑制

時間経過 → 恐怖の減衰
    ↓
【Phase E】形式的な対策は継続（マスク）
しかし実質的な対策は緩和（距離、手洗い）
    ↓
インフルエンザ回復（2023年）
```

→ **恐怖心が行動の動機であり、行動そのものは結果**

## 4. Phase Eから学んだこと

### 4.1 「動機」>「行動そのもの」

**重要な発見**:
- インフルエンザ抑制に重要なのは「マスク着用」という行動そのものではない
- **「なぜマスクをするか」という動機**が重要
- 恐怖心から生まれる厳格な対策が効果的
- 習慣的なマスク着用（恐怖なし）は効果が限定的

### 4.2 社会的同調圧力の再解釈

**当初の単純な仮説**:
```
隣の人の行動 → 自分の行動
```

**より精緻な理解**:
```
【ステップ1】恐怖の社会的伝播
周囲の人々の恐怖 → 自分の恐怖（社会的感情伝染）

【ステップ2】行動の同調
恐怖を感じた人々の行動 → 自分の行動（行動同調）

【ステップ3】規範の形成
みんなが対策している → 社会規範の確立

【ステップ4】規範の弱体化
恐怖の減衰 → 規範の守られなくなる
```

→ **「隣の人」は重要だが、恐怖心という感情が先にある**

### 4.3 Google Trendsの限界と可能性

**限界**:
- 検索行動≠実際の行動
- 習慣化後は検索されない
- 行動の「質」や「厳格さ」は測定できない

**可能性**:
- 関心が高い時期の行動の「導入」を捉える
- 恐怖心などの感情的要因の測定には有効
- 先行指標としての価値（検索→行動）

### 4.4 複雑性の認識

**Phase C・D・Eでも92.2%は未説明**:

複合モデル（死亡数 + 恐怖指数 + 隣の人指数）でもR²=7.8%に留まります。

**他の重要な要因**:
1. **政策介入**: 緊急事態宣言、まん延防止措置、学校閉鎖
2. **医療システム**: インフルエンザ検査の優先度低下、受診控え
3. **ウイルス学的要因**: 国際的なウイルス流動の停止、株の競合
4. **社会経済**: リモートワーク、外出頻度の減少、経済活動の縮小
5. **季節性とタイミング**: COVID-19流行とインフルエンザシーズンの重複

→ **単一の心理的・行動的要因では説明できない複雑な現象**

## 5. 学術的・実務的な価値

### 5.1 学術的な貢献

**1. 仮説の否定も価値ある発見**:
- Phase Eは当初の仮説を支持しなかった
- しかし、これは重要な科学的発見
- 直感的に正しそうな仮説でも検証が必要

**2. 行動経済学・社会心理学への示唆**:
- 行動変容において「動機」が「行動」より重要
- 社会的同調は恐怖という感情を介する
- 外発的動機（規範）より内発的動機（恐怖）が強力

**3. 測定方法論への貢献**:
- Google Trendsの可能性と限界の明確化
- 代理指標の選択の重要性
- 感情的要因 vs 行動的要因の測定の違い

### 5.2 実務への応用（薬局DX）

**1. 恐怖指数を主要指標として採用**:
- Phase Eの結果、Phase D（恐怖指数）が最良と確認
- Google Trends「コロナ 死亡」「医療崩壊」等を監視
- マスク着用率などの行動指標より信頼性が高い

**2. 行動指標の限界を認識**:
- マスク着用率データがあっても、それは遅行指標
- 恐怖心→行動の因果順序を考慮
- 恐怖指数の方が先行性がある

**3. 持続的な行動変容の難しさ**:
- 恐怖心は時間とともに減衰
- 規範化だけでは不十分
- ポジティブな動機づけ（健康、家族を守る等）が必要

### 5.3 公衆衛生政策への示唆

**1. 恐怖に頼らない啓発の重要性**:
- Phase Eの結果、恐怖心の減衰は避けられない
- 初期: 恐怖心が高い時期 → 正確な情報提供、過度な不安の緩和
- 中期: 慣れが生じる時期 → 行動の習慣化・内発的動機づけ
- 後期: 恐怖が薄れた時期 → ポジティブな規範形成

**2. 多面的なアプローチの必要性**:
- 恐怖指数でも5.7%の説明力に留まる
- 92.2%は未説明 → 単一のアプローチでは不十分
- 政策、医療体制、社会規範の統合的な対策が必要

## 6. Phase Eの位置づけ

### 6.1 プロジェクト全体の中での役割

```
Phase A（回帰予測）
    → Phase B（因果推論の試み）→ 失敗
        → Phase C（客観的脅威: 死亡数）→ R²=2.4%
            → Phase D（主観的恐怖: 検索トレンド）→ R²=5.7%（最良）
                → Phase E（行動指標: マスク関連）→ R²=1.2%（最低）
                    ↓
            【統合的理解】
            恐怖心が行動の動機であり、
            行動そのものではインフルエンザ抑制を説明できない
```

### 6.2 Phase Eの成功と失敗

**失敗**:
- 仮説（R²=30～50%）は支持されなかった
- 隣の人指数は3つの中で最も説明力が低い

**成功**:
- 直感的な仮説を科学的に検証
- Phase Dの優位性を再確認
- 因果メカニズムの理解を深化
- 恐怖→行動の経路の重要性を証明

→ **「期待した結果が得られなかったこと」自体が価値ある発見**

### 6.3 Phase BとPhase Eの共通点

**Phase B（因果推論の失敗）**:
- 反事実的予測を試みるも、ラグ特徴量の循環参照問題
- 結果: -1.80%の抑制率（予想と逆）
- 学び: 単純なMLでは因果推論は不可能

**Phase E（社会的同調仮説の否定）**:
- 「隣の人指数」が最良と予想するも、最も低い説明力
- 結果: R²=1.22%（予想の1/25）
- 学び: 行動より動機が重要

**共通点**:
- 両者とも仮説が否定された
- しかし、失敗から重要な洞察を得た
- 科学的探究のプロセスの価値を示す

## 7. 今後の研究課題

### 7.1 マスク着用率の実測データとの比較

**提案**:
- Google検索データではなく、実際のマスク着用率データ（調査データ等）を使用
- 着用率の「有無」だけでなく「場面」（屋外・屋内、混雑時等）を区別
- 着用の「厳格さ」（鼻まで覆っているか等）も測定

### 7.2 質的データの統合

**提案**:
- アンケート調査: 「なぜマスクをするか」の動機を直接測定
- インタビュー調査: 行動変容のプロセスを詳細に把握
- 日記調査: 時間経過による動機の変化を追跡

### 7.3 他の行動指標との比較

**提案**:
- 「ソーシャルディスタンス」「リモートワーク」等の検索トレンド
- 交通量データ（外出頻度の代理指標）
- 飲食店予約数（社会的接触の代理指標）

### 7.4 地域間比較

**提案**:
- 北海道 vs 東京 vs 大阪 vs 沖縄
- 都市部 vs 地方
- 同調圧力の強さの地域差を検証

## 8. 結論

Phase Eは、当初の仮説「隣の人の行動が全て」を支持しませんでした。しかし、この「失敗」から以下の重要な学びを得ました：

1. **恐怖心が行動の動機であり、行動そのものは結果である**
2. **行動の「質」（厳格さ）が重要であり、「有無」だけでは不十分**
3. **社会的同調は感情伝染を介して機能する**
4. **Google Trendsは感情測定に有効だが、行動測定には限界がある**
5. **仮説の否定も科学的探究の重要な成果である**

Phase D（恐怖指数）がPhase C（死亡数）とPhase E（行動指数）より優れていることが再確認され、**「認知された危険度」が行動変容の鍵**であるという理解が強化されました。

この5段階の分析プロセス（Phase A→B→C→D→E）は、反復的な科学的探究の価値を示しています。Phase BとPhase Eという2つの「失敗」があったからこそ、より深い理解に到達できました。

---

**次のステップ**:
- 複合モデルの92.2%未説明部分の探索
- より高度な因果推論手法（Granger因果性、合成コントロール法等）の適用
- 実務への応用（60店舗薬局DXでの需要予測システム構築）

**最終的な洞察**:
> インフルエンザの99%抑制は、「隣の人がマスクをしているから自分もする」という単純な同調ではなく、「周囲が恐れている→自分も恐れる→包括的な対策を厳格に実行する」という感情伝染と行動変容の複雑なプロセスによって説明される。
